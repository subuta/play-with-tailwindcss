# play-with-tailwindcss

Tailwind CSS + Parcel tutorial.

### Prerequisites

- Node.js v10.16.3

### How to develop

```bash
# Install dependencies
npm i

# Start storybook
npm run storybook

# Start dev-server
npm run dev

# Run JavaScript Standard Style lint(with autofix)
npm run lint

# Build for production
npm run build

# Serve dist files (for production)
npm start
```
